About Game Theory

Game theory is a branch of mathematics that concerns the ways in which interacting choices of characters produce outcomes with respect to the preferences (or utilities) of those characters, where the outcomes in question might have been intended by none of the characters. A few examples of games that might be analyzed using game theory include Tic-tac-toe, Checkers, and Go. Game theory is mainly used in economics, political science, and psychology, as well as logic and computer science.
